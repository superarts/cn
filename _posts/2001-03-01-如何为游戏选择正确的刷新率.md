# 如何为游戏选择正确的刷新率

关于刷新率的问题的确是老生常谈，但是根据在网吧和学生宿舍里看到的情况，我觉得还是有必要说一下，以引起大家足够的重视。

大家都知道，我这里说的刷新率，指的是帧刷新率，通俗地说，也就是显示器在一秒钟内闪动多少回（其单位Hz就是“每秒”的意思）。刷新率过低，会使人的眼睛容易疲劳，甚至引起头痛。但遗憾的是，大多数刚刚接触电脑的人认识不到这个问题的重要性。因为电视的刷新率是60Hz（P 制，N 制是50Hz，照这么说P 制比N 制好：），所以看惯电视的人就感觉不到电脑的这个优越性，以致常常使电脑工作在低刷新率下。但是请注意，无论你是不是感觉得出刷新率不同所引起的区别，其影响是客观存在的，电脑显示器离人眼很近，低刷新率一定会引起视觉疲劳，进而会引起假性近视。其实分辨刷新率很简单，一是让显示器显示白色，二是用余光看，很容易就可以看出显示器闪还是不闪。

下面介绍如何调整刷新率。

其实相信很多人都知道，但是还是要介绍一下，原因一会就清楚了。`^^`在桌面上单击鼠标右键，在弹出菜单中选择属性，选择“设置”标签，单击“高级”按钮；先选择“监视器”标签，把复选框都打上勾，单击“更改”，选择一个正确的显示器类型，如“`Super VGA 800 x 600 x 75Hz`”（更改监视器类型这一步是针对比较早的机器的）。再选择“适配器”标签，选择“刷新速度”就可以了。在这里我想提一下什么刷新率合适的问题，可能与主流观点不相符^^。我的意见是，如果你的显示器允许，不要选择“优化”或“85HZ”，就选择100 Hz。（100 Hz以上我持保留意见：）大多数人都说85Hz和100 Hz没什么打区别，考虑显示器的使用寿命，还是选85Hz好。但其一，85Hz与100 Hz的区别，就像电磁破对人体的危害一样，至今尚无定论（请不要用报纸上的所谓“美国专家”的意见打岔：），人眼分辨不出不代表客观上就没有区别，究竟有什么影响谁也不敢打保票；其二，以前也有人说75Hz和85Hz没什么大区别，可现在TCO'99还是使用了85Hz的标准（TCO'97是75Hz），以我看超前一点也没什么不好；其三，说真的，显示器技术虽然变化不大，可三、五年一过，换机子绝对要换显示器！更何况也不是说上了100 Hz就一定五年内烧掉^^……纯属个人见解，反正我认为眼睛是最重要的！

如何为游戏选择正确的刷新率？如果你认为，做完上面一步就完事大吉了，那你就犯了一个常见的错误！我接触的许多人，都是新买的机器，商家为他们调好了分辨率和刷新率，于是就这么一直使下去了。但是请注意，现在的显示器，一般都是一个分辨率对应一个刷新率！也就是说，你只把800 x 600 x 16 bit调成了85Hz，虽然桌面的确是85Hz，但是如果运行了星际争霸，保证还是60Hz！你必须在桌面上把640 x 480 x 256c也调成85Hz（注意：不同颜色，即使分辨率相同，刷新率也不同）才行。（搞不好还需要运行一下游戏以使显示器“记住”，然后回到桌面再把分辨率改回去：）另外，虽然有的显示器不是这样的设定，但如果显示器设置里没有分辨率显示（Philips 里就有，别的不太清楚），而你看电视过多，分不出刷新率高低，最好还是请一位接触电脑时间较长的朋友帮你看一看，毕竟这和你的视力健康息息相关！

PS：	

一、如果你嫌反复改分辨率麻烦，可以打开注册表（`开始->运行->c:\windows\regedit.exe`）找到键`HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\Class\Display\0000\MODES`，这里的主键4、8、16、32指的是颜色位数，里面的主键xxx，yyy是分辨率。你可以在xxx，yyy里反复添加键值`RefreshRate`，其值就是该分辨率、颜色下的刷新率，注意：手动改注册表一定要慎重，刷新率填高了就麻烦了（得用安全模式什么的恢复，至于显示器会不会烧……我可没试过：）。此法不推荐Beginner使用。
